Solar Control System

Versão 1.1.9 (Build 2025.12)

O Solar Control System é uma plataforma de supervisão e controle de alta performance desenvolvida para plantas heliotérmicas. O sistema integra monitoramento em tempo real de sensores térmicos, estações meteorológicas e atuadores de campo, garantindo eficiência energética e segurança operacional.

Desenvolvido através da cooperação técnica entre Metalwize e USP, este software utiliza arquitetura modular para controle preciso de heliostatos e microturbina.

Backend Core Python / Flask / SCADA Br
Comunicação Modbus TCP/IP
Interface HTML5 / JS (ECMAScript 6)
Banco de Dados SQLite / PostgreeSQL
